Project Overview

Yash Jewelers is a modern jewellery ecommerce website developed using ASP.NET Core MVC. The project includes product management, shopping cart, authentication, SQL Server database, responsive design and admin panel.

Key Features

  • Product Management
  • Shopping Cart
  • User Login
  • Admin Panel
  • Responsive UI
  • SQL Server Database
  • Bootstrap 5
  • ASP.NET Core MVC
