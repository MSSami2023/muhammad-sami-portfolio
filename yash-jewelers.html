Yash Jewelers

Modern Jewellery Ecommerce Website developed using ASP.NET Core MVC, SQL Server and Bootstrap.

Project Overview

Yash Jewelers is a responsive Jewellery Ecommerce website developed using ASP.NET Core MVC. It includes Product Management, Shopping Cart, Authentication, Admin Panel, SQL Server Database and Responsive Design.

Features

  • Responsive Design
  • Product Listing
  • Product Details
  • Shopping Cart
  • User Authentication
  • Admin Dashboard
  • Category Management
  • Brand Management
  • SQL Server Database
  • Bootstrap UI

Technology Used

ASP.NET Core MVC SQL Server Entity Framework Core Bootstrap 5 HTML5 CSS3 JavaScript
